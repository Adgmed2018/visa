# References for visa-etnografo

Esta pasta é reservada para arquivos de referência detalhados que o
visa-etnografo pode precisar carregar dinamicamente conforme o contexto.

Arquivos típicos que podem viver aqui:
- Templates de output específicos
- Catálogos de referência (ex: padrões de design, vocabulário de domínio)
- Heurísticas detalhadas que ficariam longas demais no SKILL.md principal

A SKILL.md principal deste agente é auto-contida — você não precisa de
arquivos desta pasta para executar o trabalho do agente. Esta pasta
existe para permitir extensão futura sem editar o SKILL.md.
