## Resumen

Visa es un framework de **descubrimiento forward de especificación** — coordina agentes de IA para convertir conversaciones vagas, hipótesis de mercado y entrevistas en especificaciones canónicas, ejecutables y rastreables. Es el complemento *forward* de [Reversa](https://github.com/sandeco/reversa).

## Instalación rápida

```bash
pip install visa-sdd
visa install
# Abrir el agente de IA (Claude Code, Antigravity, Cursor, etc.) y ejecutar /visa
```
